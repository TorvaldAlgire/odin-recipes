# odin-recipes

An Odin Project practice project recipe website.

Project Overview

This project is part of The Odin Project Foundations course. The goal of the assignment was to build a simple recipe website using basic HTML concepts. The site contains multiple recipe pages connected through links and demonstrates the use of common HTML elements such as headings, paragraphs, lists, images, and links.

The recipes included in this project are:

Shepherd's Pie

Spaghetti Carbonara

Chicken Vegetable Stir Fry

Each recipe page includes a description, an ingredient list, and cooking instructions.

Skills Practiced
HTML Structure

Through this project I practiced building basic HTML page structure and using semantic elements such as:

Headings (h1, h2)

Paragraphs (p)

Unordered lists (ul)

Ordered lists (ol)

Images (img)

This helped reinforce how information is organized and displayed on a webpage.

Lists and Content Organization

I gained a strong understanding of how ordered and unordered lists work in HTML. These were used to organize ingredients and cooking steps in a clear and readable format.

HTML Attributes

I practiced using HTML attributes, particularly with the image tag. I standardized the size of recipe images by setting them to 300x300 pixels, which helped keep the layout consistent across all recipe pages.

Relative Paths

One of the most important concepts I reinforced during this project was relative file paths. I debugged my first issue related to incorrect paths while linking images and pages. Fixing this helped me better understand how files reference each other within a project directory.

Working in the Terminal

This project also helped me become more comfortable using the terminal. I practiced:

Navigating directories

Moving and copying files

Transferring images between my Ubuntu filesystem and my desktop filesystem

These tasks helped improve my workflow when managing project files.
